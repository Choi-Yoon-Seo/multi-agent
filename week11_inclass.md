## 내 에이전트 주제
- 에이전트 이름: 취향 기반 플레이리스트 에이전트
- 해결하려는 문제: 사용자가 상황과 선호하는 아티스트/장르를 입력하면, 
               그에 맞는 플레이리스트 초안을 생성하는 에이전트
- 입력: 상황, 장르, 아티스트
- 출력: 플레이리스트 초안

## 일정공지 예시에서 바꾼 점
|         일정공지 예시        |                  내 에이전트                |
|---------------------------|------------------------------------------|
|           공지문            |     사용자가 입력한 상황·장르·아티스트 텍스트      |
|      날짜/대상/준비물 추출     |             상황·장르·아티스트 추출            |
|          학년별 분류         | 상황 → BPM·에너지·발렌스·인스트루멘탈 속성으로 분류 |
|        사용자별 안내문        |      도입→전개→정점→마무리 플레이리스트 초안      |
