# Week 8. 중간평가

## 평가 목표

- 학생이 1~7주차의 핵심 개념을 실제로 적용할 수 있는지 확인한다
- 암기보다 구현과 설명 능력을 본다

---

## 평가 범위

- AI 협업의 기본 원칙
- 재현 가능한 개발 환경
- MCP와 최소 MCP 서버
- 도구 호출 에이전트
- LangGraph의 상태, 분기, 반복
- 미니 프로젝트 1에서 다룬 구현 경험

---

## 권장 평가 형식

### 1. 실기 평가

아래 중 한 가지 유형으로 구성한다.

- 주어진 외부 API를 간단한 MCP 도구로 래핑하기
- 기존 도구 호출 코드를 수정하여 입력 검증 추가하기
- LangGraph 워크플로우에 종료 조건 추가하기

### 2. 짧은 구두 설명 또는 서술형

- 왜 `.env`가 필요한가
- 왜 종료 조건이 없는 루프가 위험한가
- 도구 설명이 모호하면 어떤 문제가 생기는가

---

## 시간 운영 예시

- 20분: 안내 및 환경 점검
- 60분: 실기
- 10분: 제출 점검

## 채점 기준

- 실행 가능성
- 요구사항 충족 여부
- 에러 처리와 검증 여부
- 코드 설명 가능성
