# Playwright Check Skill

## 목표

Playwright CLI로 HTML 또는 로컬 페이지가 열리는지 확인한다.

## 기본 명령

```bash
playwright-cli open docs/week-10.html
```

## 작업 원칙

- Playwright MCP를 설정하지 않는다.
- 브라우저 확인이 필요한 경우에만 사용한다.
- 화면 확인 결과를 한두 문장으로 보고한다.
