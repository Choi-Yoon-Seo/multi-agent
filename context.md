# 취향 기반 플레이리스트 에이전트 프로젝트

## 목표
현재 상황 한 줄과 선호 장르·아티스트 필터를 입력받아, 플레이리스트 초안을 순차적으로 생성하고
사람 승인을 거쳐 Spotify 저장 준비 완료 상태로 만든다.

## 입력
현재 상황 (드라이브·집중·슬플 때 등) + 선호 장르·아티스트 필터.

## 출력
순서 확정된 플레이리스트 초안 10~20곡 및 각 곡의 선택 이유 한 줄 요약.

## 에이전트 역할
1. 상황 분석 에이전트: 입력된 상황을 BPM 
   범위·에너지·발렌스·인스트루멘탈 여부로 변환한다.
2. 필터 병합 에이전트: 사람이 입력한 장르·아티스트 필터와 
   상황 속성을 교집합으로 병합해 Spotify 
   검색 파라미터를 생성한다.
3. 플레이리스트 구성 에이전트: 후보곡 20~30개를 받아 
   도입→전개→정점→마무리 흐름으로 순서를 배치하고 
   초안을 만든다.

## 제약
- 처음에는 Python 기본 코드로 만든다.
- 코드는 `my_agent.py` 하나에서 시작한다.
- 실행 결과는 터미널에 출력하고, 나중에 `output.md`로 
  저장한다.
- 사용자 필터(장르·아티스트)가 없으면 필터 병합 에이전트를 
  실행하지 않는다.
